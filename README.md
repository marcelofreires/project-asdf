# project-asdf

A component library example for React Native

## Installation

```sh
yarn add project-asdf
```

## Usage

```jsx
import { Button } from "project-asdf";

const MyComponent = () => (
  <Button label="Press me" onPress={() => console.log('Pressed')} />
);

export default MyComponent;
```
